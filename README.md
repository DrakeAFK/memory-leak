# MEMORY_LEAK

An ultra-minimalist, high-performance JS memory game with an "Obsidian" terminal aesthetic.

## Features
- **Obsidian Theme**: Pure black and bone white minimalist UI.
- **HJKL Support**: Vim-style keyboard mappings for high-speed play.
- **Precision Telemetry**: Real-time tracking of latency and throughput (BPS).
- **Void Centric Design**: Zero-distraction interface for maximum focus.

## Controls
- **Enter**: Initialize Session
- **H**: Green
- **J**: Red
- **K**: Yellow
- **L**: Blue

## Inspired by
Designed for the multimillionaire engineer who values precision and minimalism. Terminal aesthetics meet high-speed memory training.
